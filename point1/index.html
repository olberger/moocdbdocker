Sujet du projet

Utilisation de Docker pour la mise en place de MOOC

Mise en place d'un suivi des TP

  • Construit sur les Linux Containers
  • Permet de déployer des applications avec leur environnement, exécutables sur tous les noyaux Linux récents
  • Solution exitante pour les autres OS : machine virtuelle Linux

Contraintes des environnements de MOOC

  • Déployables sur les postes utilisateurs, et éventuellement sur serveur
  • Facilité d'installation et de mise à jour
  • Compatibilité large

Prototype

Équivalent de la VM Vagrant utilisée pour le MOOC de bases de données

  • Utilisation de deux conteneurs Docker :
    • Un conteneur PostgreSQL
    • Un conteneur Apache avec PhpPgAdmin et l'application développée pour le MOOC

Performances

  • Occupation mémoire presque identique (sauf sous Linux)
  • Taille de la VM Virtual Box exportée :
    • Vagrant : 307 Mo
    • Docker : 222 Mo

Tracking des actions

  • Compatibilité avec Moodle (utilisé pour les MOOC à TSP)
  • Mode hors-ligne
  • Protection de la vie privée

Tracking des actions

  • Utilisation d'un agent sur chaque environnement de travail
    • Parsing des logs
    • Scripts js
    • etc.
  • Centralisation des événements
  • Injection dans Moodle via un plugin
